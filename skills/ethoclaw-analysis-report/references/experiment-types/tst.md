# TST

`TST` 指悬尾测试（Tail Suspension Test）。

## 主要评估什么

- 对不可逃避应激的行为反应
- 常用于抗抑郁样药效筛查

## 常见实验流程

- 将小鼠尾部固定后悬挂
- 记录固定时长内的挣扎与静止
- 常见时长为 6 分钟

## 常见 readouts

- 静止时间（immobility time）
- 静止潜伏期（latency to immobility）
- 活动/挣扎时间（mobility / struggling time）

## 写报告时优先怎么总结

- 直接总结静止时间是增加还是减少
- 如果有 latency 或 struggling time，也直接写出主动应对或被动应对的方向
- 单样本项目可概括“以静止为主”或“挣扎较多”

## 推荐总结方式

- “静止时间增加，提示被动应对（passive coping）增强。”
- “静止时间减少，同时挣扎时间增加，表现为更主动的应激应对。”

## 不要越界

- 不要把 TST 结果直接写成“患抑郁”或“抑郁程度更高”
- 不要忽略运动能力或镇静效应对 immobility 的影响
