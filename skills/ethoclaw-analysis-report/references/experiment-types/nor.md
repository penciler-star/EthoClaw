# NOR

`NOR` 指新物体识别测试（Novel Object Recognition）。

## 主要评估什么

- 识别记忆（recognition memory）
- 新奇偏好（novelty preference）

## 常见实验流程

通常包括两个核心阶段：

1. 熟悉阶段（familiarization / training）：动物接触两个相同或等价的物体
2. 测试阶段（test）：将其中一个物体替换为新物体（novel object），记录探索偏好

## 常见 readouts

- 新物体探索时间（novel object exploration time）
- 熟悉物体探索时间（familiar object exploration time）
- 判别指数（discrimination index）
- 偏好指数（preference index）
- 总探索时间（total object exploration time）

## 写报告时优先怎么总结

- 先总结对新物体的偏好是否增强、减弱或不明显
- 如果有判别指数或偏好指数，直接写出方向
- 单样本项目可以直接概括“偏向新物体 / 偏向熟悉物体 / 偏好不明显”

## 推荐总结方式

- “对新物体的探索时间更长，表现出明显的新奇偏好（novelty preference）。”
- “新物体偏好减弱，判别指数下降。”

## 不要越界

- 不要把 NOR 单独写成“学习记忆全面受损/恢复”
- 不要忽略物体本身吸引力差异与侧偏好（side bias）
