# OFT

`OFT` 指开放场测试（Open Field Test）。

## 主要评估什么

- 自发活动（locomotor activity）
- 新环境探索（novelty-driven exploration）
- 焦虑样行为的部分指标，例如中心区回避或贴边行为

## 常见实验流程

- 将动物放入开放场（正方形或圆形 arena）
- 在固定时长内记录运动轨迹和行为
- 常把场地分为中心区（center zone）与边缘区（periphery）

## 常见 readouts

- 总路程（total distance traveled）
- 平均速度（velocity）
- 中心区停留时间（time in center）
- 中心区进入次数（center entries）
- 贴边行为（thigmotaxis）
- 竖立（rearing）
- 静止时间（immobility）

## 写报告时优先怎么总结

- 先总结活动量是增加、减少还是基本稳定
- 再总结中心区探索和边缘区偏好是否变化
- 如果总路程和中心区探索方向一致，可以直接写出“活动下降并伴随中心探索减少”这类结论
- 单样本项目可以直接概括“高活动 / 低活动”“中心探索多 / 少”“贴边明显 / 不明显”

## 推荐总结方式

- “总活动量（locomotor activity）下降，同时中心区探索减少。”
- “受试动物更多停留在边缘区，表现出明显的贴边行为（thigmotaxis）。”
- “中心区进入次数增加，提示开放场中的中心探索增强。”

## 不要越界

- 不要把 center time 的变化直接写成焦虑水平变化，而完全忽略活动量
- 不要把 OFT 单独写成抑郁样结论
