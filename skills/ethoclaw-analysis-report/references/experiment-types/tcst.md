# TCST

`TCST` 在本 skill 中指三箱社交测试（Three-Chamber Social Test / Three-Chamber Sociability and Social Novelty Test）。

## 主要评估什么

- 社交趋近（sociability）
- 社交新异偏好（social novelty preference）
- 在部分设计下，也可反映社交识别（social recognition）

## 常见实验流程

典型流程分为 2 到 3 个阶段：

1. 适应阶段：受试鼠自由探索三箱装置
2. 社交趋近阶段：一侧放陌生鼠（stranger mouse），另一侧放空杯或空笼（empty cup / empty cage）
3. 社交新异阶段：一侧放熟悉鼠（familiar mouse），另一侧放新的陌生鼠（novel stranger）

## 常见 readouts

- 各箱停留时间
- 靠近刺激杯/笼的时间
- 嗅探（sniffing）时间
- 各箱进入次数
- 社交偏好指数（social preference index）
- 新异偏好指数（social novelty index）

## 写报告时优先怎么总结

- 如果数据来自“陌生鼠 vs 空杯”阶段，优先总结社交趋近是否增强或减弱
- 如果数据来自“熟悉鼠 vs 新陌生鼠”阶段，优先总结社交新异偏好是否存在
- 如果只有 chamber time，也可以直接总结空间偏好和社交线索偏好方向
- 单样本项目可以直接写出“该样本主要停留在哪一侧、接近哪类刺激更多”

## 推荐总结方式

- “在社交趋近阶段（sociability phase），受试鼠更长时间停留在包含陌生鼠的一侧，提示社交趋近较强。”
- “在社交新异阶段（social novelty phase），受试鼠对新陌生鼠的接近更多，表现出社交新异偏好。”
- “当前单样本结果显示，该样本主要活动于……区域，对……刺激的接近更多。”

## 不要越界

- 不要把 chamber time 的差异直接写成“社交记忆受损”，除非设计和指标足够支持
- 不要把低活动量样本自动解释为社交回避
- 不要把单次 TCST 结果写成完整的自闭样行为结论
